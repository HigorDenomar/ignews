## Como usar

### Clone o projeto:

```sh

git clone https://github.com/HigorDenomar/ignews.git

```

### Rode o projeto (npm)

```sh
# Entre na pasta do projeto
cd ignews

# Instale as dependências
npm install

# Rode o projeto
npm run dev

```

#### Ou, se preferir o yarn:

```sh
# Entre na pasta do projeto
cd ignews

# Instale as dependências
yarn

# Rode o projeto
yarn dev

```

O projeto poderá ser acessado no seu [localhost:3000](http://localhost:3000)
